Bước 1: Start Metro bundler

CLI sẽ chạy Metro bundler (server JS) để:

Biên dịch file JS (App.js, index.js) sang bundle.

Cung cấp bundle cho ứng dụng Android/iOS.

Bước 2: Build native code

React Native CLI gọi Gradle (Android) hoặc Xcode build (iOS):

Android: gradlew assembleDebug

iOS: xcodebuild -scheme MyApp

Build file native bao gồm:

Java/Kotlin (Android)

Objective-C/Swift (iOS)

Các dependencies native (các library có .aar hoặc .framework)

Bước 3: Cài đặt app trên thiết bị

CLI sẽ tự động install APK (Android) hoặc app (iOS) trên thiết bị/emulator.

Bước 4: Kết nối JS với native

App native khởi động và load bundle JS từ Metro bundler.

JS code chạy trong JavaScriptCore (iOS) hoặc Hermes/JS engine (Android)

Native và JS giao tiếp qua bridge:

JS gửi event -> native

Native gửi event -> JS
